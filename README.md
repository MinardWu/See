# 项目描述
“初见”是一款锁屏应用，但与其他锁屏应用的不同之处在于加入了社交元素。用户之间都是一对一的好友关系，用户能够拍照、上传图片，并立即作为好友的锁屏，好友打开手机便能第一时间看到。
